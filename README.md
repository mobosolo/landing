# Landing Page MealFlavor

## Fichiers
- `index.html`
- `styles.css`

## Etapes rapides
1. Remplace `YOUR_APK_LINK_HERE` dans `index.html` par le lien direct vers ton APK.
2. Ouvre `index.html` dans un navigateur pour verifier.

## Hebergement simple (3 options)

### Option A - Netlify (recommande)
1. Va sur Netlify et "Add new site".
2. Glisse le dossier `landing/` dans l'interface.
3. Netlify te donne un lien public.

### Option B - Vercel
1. Cree un nouveau projet.
2. Importe ce dossier `landing/`.
3. Deploie.

### Option C - GitHub Pages
1. Cree un repo `mealflavor-landing`.
2. Pousse `landing/index.html` et `landing/styles.css`.
3. Active GitHub Pages.

## APK
Tu peux heberger ton APK sur:
- Google Drive (lien public direct)
- Dropbox
- Firebase Storage
- GitHub Releases
